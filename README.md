# Klipper fork  various printers

Fork of [Klipper3d/klipper](https://github.com/Klipper3d/klipper).

## Available branches

- `master` — upstream mirror, not for Snapmaker J1!
- `snapmakerj1` — Snapmaker J1 (GD32F307)
- `snapmakerj1-debug` — same as above, with verbose debug output
- `ffcp2` — Support for Flashforge Creator Pro 2 (Unmaintained)
- `ffcp2_basic` — same as above, less advanced (Unmaintained)
